# Clubmanagement: Customizations for Swiss Football Clubs
This is an Odoo Addon that enlarges the Clubmanagement Addon for Clubs within the Swiss Football Association (SFV).

## Requirements
This Addon requires the following addons:
* *clubmanagement*: Base Addon to manage clubs within Odoo
* *l10n_ch_clubmanagement*: Switzerland Customizations for Clubmanagement
